# Subtitle Annotation Tool

A simple web-based tool for tagging subtitle files with character names.

## What It Does

Load subtitle files (.srt or .ass format) and quickly annotate each line with the character who's speaking. Perfect for extracting character-specific dialogue from TV shows, anime, or movies.

## How to Use

1. Open the HTML file in your web browser
2. Load your subtitle file
3. Tag each line with a character name (or skip it)
4. Export your annotated subtitles

## Requirements

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A subtitle file (.srt or .ass format)

## Features

- Works entirely in your browser - no installation or server needed
- Keyboard shortcuts for fast tagging
- Save your progress and resume later
- Export in multiple formats

## Getting Started

Simply open `index.html` in your browser and load a subtitle file to begin annotating.
